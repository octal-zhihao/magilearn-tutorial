Welcome to Magilearn's documentation!
===================================

MagiLearn 是一个轻量级的自定义机器学习库，支持特征选择、模型选择、数据预处理、分类、回归、聚类等模块。通过模块化设计，MagiLearn
便于扩展并易于集成在机器学习项目中。

点击 :doc:`usage` 以了解更多信息。

.. note::

   本项目还在积极建设中。
